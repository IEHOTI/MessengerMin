# Локальный мессенджер Min

Простой локальный мессенджер с поддержкой передачи файлов, написанный на C++ с использованием Qt.

## Описание

Min - это локальный мессенджер для обмена сообщениями и файлами между устройствами в одной сети. Приложение автоматически обнаруживает другие устройства в сети и позволяет установить соединение для общения.

## Основные возможности

- **Автоматическое обнаружение устройств** в локальной сети (broadcast UDP)
- **Текстовый чат** между двумя устройствами
- **Передача файлов** с подтверждением и прогрессом
- **Интуитивный интерфейс** с панелями устройств и чата
- **Локальная работа** - данные не покидают вашу сеть
- **Быстрое соединение** через TCP

## Технологии

- **Язык**: C++ 17
- **Фреймворк**: Qt 6 (Widgets, Network)
- **Сборка**: CMake
- **Протоколы**: TCP для сообщений, UDP для обнаружения
- **Форматы данных**: JSON для сообщений

## Сборка и запуск

### Требования
- Qt 6 (с модулями Widgets и Network)
- CMake 3.16+
- Компилятор с поддержкой C++17

### Сборка
mkdir build
cd build
cmake ..
cmake --build .

### Запуск
./MessengerMin  # Linux/Mac
MessengerMin.exe  # Windows
